ИНСТРУКЦИЯ ДЛЯ ОБНОВЛЕНИЯ ПРОДАКШН

Текущая ситуация:
- Старые процессы app.py работают под root на порту 5000
- Новая версия с исправлениями запущена на порту 5003
- Нужно обновить nginx конфигурацию

Команды для выполнения (требуют sudo):

1. Остановить старые процессы:
   sudo pkill -f "python.*app.py"

2. Обновить nginx конфигурацию:
   sudo sed -i 's/proxy_pass http:\/\/127\.0\.0\.1:5000;/proxy_pass http:\/\/127\.0\.0\.1:5003;/' /etc/nginx/sites-available/cash-shifts

3. Перезагрузить nginx:
   sudo systemctl reload nginx

4. Проверить результат:
   curl https://cashshifts.tsyndra.ru

Альтернативно - можно просто перезапустить systemd сервис:
   sudo systemctl restart cash-shifts
